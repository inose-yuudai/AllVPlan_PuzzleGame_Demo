# AllVPlan_PuzzleGame_Demo
---

## 1. 概要

### 友人たちと開発しているゲーム「」の自身が代表して担当している部分のみを書き出したデモゲームです。

このゲームでは架空のVtuber「境目コエル」と共に配信と呼ばれるミニゲームを通してチャンネル登録者やお金を増やしていく育成型のゲームです。
このリポジトリではそのミニゲームの一つだけをプレイできる状態です。

## 1. ゲームルール

- D&D、もしくはSpaceキーを押してから矢印キーでパズルを移動させて、位置を入れ替えることができます。

- Eキーを押すと３つ揃っている箇所が消えます。1回で何個消えたかを「チェイン」として数えられます。落ちてきて揃ったものは「コンボ」となります。チェインもコンボも大きな得点になります。

- コンボやチェインを重ねると、コメント欄に揃えた絵文字がスタンプ(Youtubeのメンバーシップ限定スタンプを参考)として送られます。スタンプが多いとコエルちゃんのテンションが上がり、登録者や視聴者数が増えやすくなります。

- 90秒以内になるべく多くの登録者や視聴者数を増やすことが目的です。


## 2.　ゲーム案について
配信としては歌配信でしたので音ゲーを作ろうとしていましたが結果的にはパズルゲームになりました。
届けたい体験としてはプレイヤーはVtuberのモデレーターになって歌配信を盛り上げる手助けをするだったので
自身のプレイがすぐフィードバックとして帰ってくるゲームを目指した。
既存のパズルゲームを分析し、コンボの気持ちよさと焦らずに体験できることを目指してルールを制作

## 3.　実装について

最初は全く別のルールで作成したのでかなり紆余曲折しました。
